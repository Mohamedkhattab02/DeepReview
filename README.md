📚 DeepReview
מערכת Web חכמה לניתוח והבנה של מאמרים אקדמיים באמצעות בינה מלאכותית

🎯 סקירה כללית
DeepReview היא מערכת מתקדמת המיועדת לסטודנטים וחוקרים המעוניינים לשפר את תהליך הקריאה וההבנה של מאמרים אקדמיים. המערכת משלבת כלי בינה מלאכותית מתקדמים לניתוח תוכן, יצירת תקצירים, שיחה אינטראקטיבית ולמידה מעמיקה.

✨ תכונות עיקריות
👨‍🎓 לסטודנטים:

📤 העלאת מאמרים - העלאת קבצי PDF וניתוח אוטומטי
🤖 צ'אט אקדמי חכם - שיחה אינטראקטיבית עם בוט מבוסס LLM על תוכן המאמר
💭 בוט סוקרטי - למידה מונחית באמצעות שאלות הדרגתיות ומשוב חכם
🔄 השוואת מאמרים - השוואה בין שני מאמרים לפי נושאים, שיטות ותוצאות
📊 מעקב התקדמות - גרפים וסטטיסטיקות אישיות
📥 ייצוא - ייצוא תקצירים ותובנות ל-PDF או Word

👨‍🏫 למרצים:

📋 פאנל ניהול - צפייה ברשימת סטודנטים וניתוח התקדמות
📈 דוחות וניתוחים - הפקת דוחות על איכות הבנה וחשיבה ביקורתית
📊 Analytics - גרפים ומגמות של ביצועי הסטודנטים
🗂️ ניהול מאמרים - צפייה, חיפוש ומחיקה של מאמרים במערכת


🛠️ טכנולוגיות
Frontend:

Next.js (React) עם TypeScript
Tailwind CSS לעיצוב רספונסיבי
App Router לניהול דפים

Backend:

Next.js Server Actions & API Routes
Google Gemini API לעיבוד שפה טבעית
Supabase (PostgreSQL + Auth + Storage)

Hosting:

Vercel
